# GreenRoute — Smart Low-Carbon Route Planner 🌍

GreenRoute is a climate-tech web application that helps users compare travel routes based on **time, cost, and carbon emissions** so they can choose more sustainable commuting options.

Instead of optimizing only for speed, GreenRoute helps optimize for **environmental impact** too.

## 🚀 Features

* Compare Car, Public Transport, and Bike/Walking routes
* View estimated CO₂ emissions for each route
* Identify the **Greenest Route**
* Carbon Savings Calculator
* What-If Impact Simulator
* Gemini Climate Coach (AI Assistant)
* Earth Day Pledge
* localStorage support
* Dark climate-tech UI

---

## 🧠 How It Works

Enter:

* Start location
* Destination

GreenRoute compares routes by:

* Travel time
* Cost
* Carbon emissions

Users can then choose a greener route and view:

* Carbon savings
* Annual impact
* Tree equivalents
